# Simple Django API - عرض بيانات طلاب

## 💻 كيف شغلت السيرفر؟

1 شغلت السيرفر باستخدام cmd:
   
    python manage.py runserver
   


## 🌐 كيف جربت الـ API؟

1. دخلت الرابط في المتصفح:
    
    http://127.0.0.1:8000/students
    
2. ظهرت لي قائمة الطلاب، كل طالب في سطر:
    ```
    أحمد - 21 سنة - هندسة برمجيات
    سارة - 22 سنة - علوم حاسوب
    محمد - 20 سنة - ذكاء اصطناعي
    ليلى - 23 سنة - هندسة شبكات
    خالد - 24 سنة - أمن معلومات
    ```

> ملاحظة: البيانات موجودة داخل الكود، بدون قاعدة بيانات.

##  مسار الملف المسؤول عن عرض الطلاب:
- داخل `students/views.py`

from django.http import HttpResponse

def student_list_text(request):
    students = [
        {"name": "أحمد", "age": 21, "major": "هندسة برمجيات"},
        {"name": "سارة", "age": 22, "major": "علوم حاسوب"},
        {"name": "محمد", "age": 20, "major": "ذكاء اصطناعي"},
        {"name": "ليلى", "age": 23, "major": "هندسة شبكات"},
        {"name": "خالد", "age": 24, "major": "أمن معلومات"},
    ]

    result = ""
    for student in students:
        result += f"{student['name']} - {student['age']} سنة - {student['major']}\n"

    return HttpResponse(result, content_type="text/plain; charset=utf-8")
